# PSVRWatcher

## About

Generates PSVR software release info(dates, genre, rating, ...) in CSV format by scraping official web page.

## Supported OSes

 * Windows 10
 * OS X 10.11


## Prerequisites

 * .NET Core Runtime 1.0.1+

## Setup (1st time only)

```
$ cd src/PSVRWatcher/
$ dotnet restore
```

## Usage

```
$ cd src/PSVRWatcher/
$ dotnet run
```
