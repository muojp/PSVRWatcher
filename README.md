# PSVRWatcher

## About

Generates PSVR software release dates in CSV format by scraping official web page.

## Supported OSes

 * Windows 10
 * OS X 10.11


## Prerequisites

.NET Core Runtime

## Setup (1st time only)

```
$ cd src/PSVRWatcher/
$ dotnet restore
```

## Usage

```
$ cd src/PSVRWatcher/
$ dotnet run
```
